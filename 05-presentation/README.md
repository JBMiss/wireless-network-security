# 05 — Wireless Security Configurations Briefing

**Course:** CYB/360 — Wireless Security
**Deliverable:** Security Configurations for Wireless Network (presentation)

## The scenario

Synthesize the wireless security work from earlier deliverables into an executive-style briefing. The presentation pulls together the design decisions, configuration choices, and threat analysis into a single overview suitable for stakeholder communication.

## What I produced

A PowerPoint presentation ([`Security_Configurations_Compressed.pptx`](./Security_Configurations_Compressed.pptx)) covering the security configurations recommended across the wireless deployment scenarios — design choices, encryption standards, threat exposures, and countermeasures.

## What this demonstrates

- **Synthesis ability** — distilling a multi-week project into a short briefing rather than presenting every detail
- **Visual communication** — using slides effectively to support spoken explanation, not duplicate it
- **Stakeholder-aware framing** — focusing on the security posture and trade-offs, not the technical minutiae

## Key takeaway

A wireless security project isn't really finished until the people who fund or sign off on it understand it. The presentation is the deliverable that makes the technical work actionable for decision-makers.
