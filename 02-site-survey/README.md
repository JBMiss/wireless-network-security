# 02 — Residential Site Survey

**Course:** CYB/360 — Wireless Security
**Deliverable:** Site Survey for Secure Wireless Home Network
**Scenario:** Multi-story home with 10+ devices including smart home and media streaming

## The scenario

Conduct a site survey for a multi-story residence and recommend a secure wireless network upgrade. The home requirements:

- 10+ connected devices including smart home equipment
- Heavy media streaming usage
- Indoor and outdoor coverage
- Compliance with current security standards
- Adaptable for future technology growth

## What I produced

A site survey report ([`Site_Survey_for_Secure_Wireless_Home_Network.docx`](./Site_Survey_for_Secure_Wireless_Home_Network.docx)) following standard survey methodology:

### Methodology
- Physical walkthrough mapping the home layout
- Identifying potential dead zones and coverage gaps
- Evaluating existing infrastructure and ISP capabilities
- Mapping RF interference sources

### Key findings
- Multiple zones with complex wiring needs (including outdoor)
- RF interference from household appliances and Bluetooth devices
- Existing infrastructure inadequate for high-bandwidth, multi-device usage
- Neighboring Wi-Fi networks on overlapping channels

### Recommendations
- High-gain omnidirectional antennas centrally placed on each floor
- Mesh extender deployment for outdoor coverage
- Channel planning to avoid neighbor interference
- Modern router supporting WPA3 and network monitoring features

## What this demonstrates

- **Survey methodology, not just opinion** — walkthrough → interference analysis → infrastructure evaluation → recommendation. This is the same workflow used in enterprise site surveys, just at home scale.
- **RF awareness** — recognizes that microwaves, Bluetooth, and neighboring networks all contribute to interference, not just "the router is bad"
- **Pragmatic recommendations** — antenna placement and channel planning are concrete actions, not vague "improve security" suggestions

## Why this matters

Site surveys are foundational to wireless work. Whether the deployment is a single-story office, a multi-floor home, a warehouse, or a hospital, the methodology is the same: **map the space, identify the interference, place equipment intentionally, validate coverage**. This deliverable shows I can run that process end-to-end.
